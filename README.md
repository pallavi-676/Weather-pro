# 🌤 Weather Pro

A sleek and modern **weather application** built with **HTML, CSS, and JavaScript**, designed to deliver real-time weather insights with an interactive and visually dynamic user experience.

---

## 🚀 Features

* 🔍 Search weather by city name
* 🌡 Real-time temperature, humidity, and wind speed
* 🌈 Dynamic UI background based on weather conditions
* 🌧 Animated rain effects for immersive experience
* 🎨 Clean glassmorphism-inspired interface
* ⚡ Lightweight and fast (no external frameworks)

---

## 🛠 Tech Stack

* **Frontend:** HTML5, CSS3 (Flexbox, Animations)
* **Logic:** Vanilla JavaScript (ES6+)
* **API:** OpenWeather API

---



## ⚙️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/pallavi-676/Weather-pro.git
```

### 2. Navigate to the project directory

```bash
cd Weather-pro
```

### 3. Add your API key

Open `script.js` and replace:

```js
const apiKey = "YOUR_API_KEY_HERE";
```

### 4. Run the application

Simply open `index.html` in your browser.
