# 🌤 Weather Pro

A modern and dynamic weather application built using **HTML, CSS, and JavaScript**.
It provides real-time weather data with a clean UI and interactive background effects.

---

## 🚀 Features

* 🔍 Search weather by city name
* 🌡 Real-time temperature, humidity, and wind speed
* 🌈 Dynamic background based on weather conditions
* 🌧 Animated rain effect for rainy weather
* 🎨 Modern glassmorphism UI
* ⚡ Fast and lightweight (no frameworks)

---


## 🛠 Tech Stack

* HTML5
* CSS3 (Flexbox + Animations)
* JavaScript (Vanilla JS)
* OpenWeather API

---

## ⚙️ Setup Instructions

1. Clone the repository:

```bash
git clone https://github.com/pallavi-676e/weather-pro.git
```

2. Navigate to project folder:

```bash
cd weather-pro
```

3. Open `script.js` and add your API key:

```js
const apiKey = "YOUR_API_KEY_HERE";
```

4. Open `index.html` in your browser.
